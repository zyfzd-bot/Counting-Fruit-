#📋 项目简介

  `数果短链`是一个高效、便捷的短链接生成与管理平台帮助用户将冗长的URL转换为简洁易记的短链接，并提供完整的数据统计功能。

✨ 核心功能

✅ 已实现功能

· 短链生成：快速将长链接转换为短链接
· 链接管理：统一的仪表板管理所有短链接
· 点击统计：实时查看每个短链的点击数据

🔜 即将上线

· 有效期设置：自定义短链接的有效期限
· 二维码生成：为每个短链接自动生成专属二维码
· 二维码扫描统计：独立统计二维码扫描数据
· 自定义二维码：支持Logo嵌入和样式定制
· 批量二维码下载：一键导出所有短链二维码

🚀 使用场景

· 社交媒体分享 - 让链接更简洁美观
· 营销活动跟踪 - 精准统计推广效果
· 内部链接管理 - 简化企业内部资源共享
· 印刷材料配套 - 结合二维码线下推广

📊 数据统计

· 实时访问数据监控
· 地域来源分析
· 设备类型统计
· 访问时间分布
· 浏览器类型统计

 # 安装教程

PHP版本：7.3 7.2

Nginx服务器首先站点部署伪静态

```nginx
location / {
    try_files $uri $uri/ /redirect.php?$query_string;
}

location ~ \.php$ {
    fastcgi_pass unix:/tmp/php-cgi-74.sock;
    fastcgi_index index.php;
    fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    include fastcgi_params;
}
